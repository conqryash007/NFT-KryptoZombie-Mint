 ## 👀Checkout the project in your local
 #### 1. Clone the repo
 #### 2. `cd` to the root directory.
 #### 3. Run `npm install` to install the dependencies of the project.
 #### 4. Run `npm start` to start the local server.
 #### 5. Access the server at `localhost:3000`
